# 지도 타일 다운로드 가이드

지도 배경(종이 지도 느낌)을 표시하려면 OpenStreetMap 타일을 다운로드해야 합니다.

## 왜 타일이 필요한가요?

현재 지도에 배경이 없는 이유는 지도 타일(배경 이미지)이 없기 때문입니다. 타일을 추가하면 구글 지도처럼 종이 지도 느낌의 배경이 표시됩니다.

## 타일 다운로드 방법

### 방법 1: 간단한 방법 (권장하지 않음 - 파일이 많음)

OpenStreetMap 타일은 매우 많은 파일(수만 개)이 필요하므로, 전체를 다운로드하는 것은 비현실적입니다.

### 방법 2: 필요한 지역만 다운로드 (권장)

사용하는 지역의 타일만 다운로드하는 방법:

1. **외부 인터넷 환경에서** 다음 도구를 사용:
   - Mobile Atlas Creator (MOBAC)
   - 또는 QGIS의 타일 다운로드 플러그인

2. **다운로드할 지역 선택:**
   - 아시아, 아프리카, 남미 등 프로젝트가 있는 지역
   - 줌 레벨: 3-10 (전체 지도 보기용)

3. **타일 저장 위치:**
   ```
   frontend/lib/leaflet/tiles/{z}/{x}/{y}.png
   ```
   예: `frontend/lib/leaflet/tiles/3/4/2.png`

### 방법 3: 간단한 배경 이미지 사용 (가장 쉬움)

전체 타일 대신 간단한 배경 이미지를 사용할 수 있습니다:

1. **전 세계 지도 이미지 다운로드:**
   - 외부 인터넷에서 "world map png" 검색
   - 또는 간단한 지도 이미지를 다운로드

2. **이미지를 배경으로 사용:**
   - `frontend/images/world-map.png`에 저장
   - 코드에서 배경 이미지로 사용

## 빠른 해결책: 간단한 배경 이미지 사용

가장 간단한 방법은 전 세계 지도 이미지를 배경으로 사용하는 것입니다.

### 단계:

1. **외부 인터넷에서 지도 이미지 다운로드:**
   - 검색어: "world map simple png" 또는 "world map outline png"
   - 권장 크기: 2000x1000 픽셀 이상
   - 스타일: 종이 지도 느낌 (색상이 부드러운 것)

2. **이미지 저장:**
   ```
   frontend/images/world-map-background.png
   ```

3. **코드 수정:**
   `gis.html` 파일에서 배경 이미지를 사용하도록 수정 (이미 적용됨)

## 현재 상태

현재 코드는 타일이 없어도 작동하도록 설정되어 있습니다:
- 마커는 정상적으로 표시됩니다
- 지도 컨트롤(줌, 이동)은 작동합니다
- 배경만 비어있습니다

## 권장 사항

내부망 환경에서는:
1. **간단한 배경 이미지 사용** (가장 쉬움)
2. 또는 **필요한 지역의 타일만 다운로드** (더 정확하지만 복잡)

전체 타일을 다운로드하는 것은 권장하지 않습니다 (파일이 너무 많음).
